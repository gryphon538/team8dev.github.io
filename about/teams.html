Paly Robotics Teams


Build

The build team's job is to construct the physical aspects of the robot. Members forge robot parts by utilizing a plethora of professional grade equipment ranging from a laser cutter to an automatic metal mill. In addition, build team members can specialize in CAD, electronics, metalworking, or pneumatics.


PR

The Public Relations team is in charve of representing Team 8 to its sponsors and the public. Members make presentations to apply for awards on behalf of the team and to raise money by finding sponsors.

Programing

Using C++ the programming team writes the code that directs what the robot does. Members program the robot so it can be driven by a person or so it can run autonomously based on input from sensors.


Animation

The animation team uses Autodesk 3D animation software to. Reade short videos. Members mainly focus on creating an animation to compete in FIRST'S annual competition.
